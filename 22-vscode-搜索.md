### 22 vscode 搜索

- global search

  - shift + command + f
    - command + 方向键上下
    - command + shift + j

- go to symbol in workspace

  - command + t

- go to symbol in file

  - command + shift + o
    - : 分类排序

- search command

  - command + shift + p

- quick file navigation

  - command + p

- switch file
  - ctrl + tab
