### 23 vscode 编码

- 修复错误

  - command + .

- 参数提示

  - command + shift + space

- 补全代码

  - command + i

- 移动行

  - option / alt + 方向键上下

- 增加空行

  - command + enter `向下加`
  - command + enter + shift `向上加`

- 复制代码

  - shift + alt + 方向键上下

- 删除一个单词

  - alt / option + delete / backspace `删除一个词，eg: getName 删除getName`
  - alt / option + ctrl + delete /backspace `删除一个单词, eg: getName 删除Name`

- 跳转到错误

  - f8
  - shift + f8

- 选中所有单词
  - command + f2

```js
function getName(name = "", age = 0) {}
getName("");

get("");

CALL_BACK;
```
