### 01 vim存活

- 插入至`光标前` => `i`
- 插入至`光标后` => `a`
- 移动
  - 下 => `j`
  - 上 => `k`
  - 左 => `h`
  - 右 => `l`
- 退出`Insert模式`
  - \<Esc>
  - `\<Ctrl> + [`  => `改键 Caps 替换成 Ctrl`

#### Tips:

> 熟练 insert 与 normal模式 `频繁切换`， 用j,k,l,h 控制方向
