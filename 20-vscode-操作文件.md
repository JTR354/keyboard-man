### 20 vscode 文件操作

- 资源区 和 编辑区 切换

  - keyboard JSON
    - show explorer shift+command+e -> ctrl + ;
    - show first editor command 1 -> ctrl + '

- 资源区 移动 和 折叠代码

  - jk -> 移动
  - hl -> 折叠

- 创建文件

  - keyboard JSON
    - explorer a
  - settings JSON
    - editor Leader + n + f
  - 插件
    - File Utils

- 创建文件夹

  - keyboard JSON
    - explorer shift + a
  - settings JSON
    - editor Leader + n + d
  - 插件
    - File Utils

- 删除文件
  - keyboard JSON
    - d
- 重命名文件
  - keyboard JSON
    - r
